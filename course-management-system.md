SWENAM COLLEGE – CampusLink LMS + Student Portal Requirements Brief

Swenam College needs a web-based system that includes:

Student Portal (user accounts + course registration)

Learning Management System (LMS) where enrolled students access courses

Integration with existing website (swenamcollege.ca) in branding and navigation

User Roles
Super Admin

Full control over system settings, users, programs, and courses.

Admin / Registrar

Manages student applications, course registrations, and academic records.

Instructor

Manages course content, assignments, quizzes, grading, and communication.

Student

Applies/registers, views courses, and accesses LMS content.

Integration with Existing Website

Current website remains the public/marketing site.

Add “Student Portal / Login” button linking to the new system.

Portal and LMS should match Swenam College branding.

Public pages (programs, admissions) stay on main site; portal is for logged-in users only.

Authentication & User Management

Secure login (email + password)

Email verification, password reset

Admin can:

Create users manually

Edit user details

Activate/deactivate accounts

Assign roles (student, instructor)

Basic user fields:

Full Name

Student ID (auto-generated)

Email

Phone

Program / Department

Intake / Semester

Student Portal & Course Registration
Student Registration Flow

Student clicks “Apply / Register” on website.

If new user → create account.

Fill registration form (personal info, program, courses, preferred intake).

Submit.

Admin / Registrar Flow

Admin sees new applications:

Review details

Approve / reject / request more information

Assign program + courses

When approved:

Courses appear in student’s My Courses area.

Student Portal After Approval

(Future: financials, documents, attendance)
